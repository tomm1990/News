[no params]

getNewsByRateBiggerThan

Show news by rate

Provides json full of data which represent all articles rated greater than the specified parameter

@type
METHOD : post

@param
rate // int value

@return
[no returns]

@i.e
https://ws17ex1goldnews.herokuapp.com/getNewsByRateBiggerThan
[rate : 8]

getNewsByTitleAndRate

Show news by title and rate

Provides json full of data which represent all articles with the specified title and rate

@type
METHOD : get

@param
title // String that specifies the article title
rate // int value

@return
[no returns]

@i.e
https://ws17ex1goldnews.herokuapp.com/getNewsByTitleAndRate/Israel/10
